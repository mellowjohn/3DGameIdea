ai-rpg-engine

These are the live tools an AI agent can call against the project and editor. Same command paths the editor uses for World Forge and scene work.

ToolWhat it does
engine_editor_live / engine_editor_statusFlip live automation on and confirm the editor bridge
engine_editor_screenshot / engine_editor_ui_query / engine_editor_inputCapture the window, find UI hotspots, click tabs and controls
engine_scene_planRoute work: scene, prefab, terrain, World Forge, Lua, UI…
engine_scene_applyPlace and move objects in the live scene
engine_terrain_apply / engine_water_applySculpt terrain, paint foliage, and place water
engine_world_forge_apply / engine_world_forge_map_viewAuthor story data and drive the Map Canvas view
engine_prefab_apply / engine_asset_applyCreate or update prefabs, materials, and catalogs
engine_lua_apply / engine_lua_callEdit gameplay scripts and call handlers without a rebuild
engine_hud_apply / engine_ui_canvas_mutate / engine_ui_stackHUD and UI canvas authoring / stack control
engine_project_validate / engine_project_gitValidate project data and sync authoring through git
